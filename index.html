Setup guide — connect your data

Step 1 — Create free Supabase account

Go to supabase.com → New project. Copy your Project URL and anon/public API key from Settings → API.

Step 2 — Create two tables

In Supabase → Table Editor, create:

items: id (int8, pk), barcode (text), name (text), description (text)

count_records: id (int8, pk), barcode (text), name (text), location (text), quantity (int4), serial (text), timestamp (timestamptz)

Step 3 — Import your CSV

In Supabase → items table → Import data → upload your CSV. Make sure your CSV has columns named barcode, name, description (or update the CONFIG at the top of this file to match your column names).

Step 4 — Update CONFIG in this file

supabaseUrl: 'https://xxxx.supabase.co',
supabaseKey: 'your-anon-key',
demoMode: false

Step 5 — Host on Vercel (free)

Drag this HTML file into vercel.com/new (or put it in a GitHub repo and connect it). You'll get a shareable HTTPS URL that works on any phone or desktop.
